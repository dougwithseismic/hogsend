This guide has moved. PostHog now has a [dedicated integration section](/docs/integrations/posthog), and the fastest path is no longer a dashboard walkthrough:

```bash
hogsend connect posthog --url https://api.your-app.com
```

One command, one browser consent click — it stores an encrypted credential on your instance and provisions the PostHog → Hogsend webhook destination this page used to set up by hand.
