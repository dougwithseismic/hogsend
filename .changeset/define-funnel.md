`defineFunnel` — funnels as a code-first, event-native primitive, plural. A funnel is authored like a journey: ordered `stages` entries (plain strings, or objects putting the semantics ON the stage — `{ id, on?, milestone? }`), where `milestone: "quoted" | "won"` picks which stages mint the money events (no `"won"` milestone = a non-monetary funnel that never mints), `on` declares the event trigger(s) that move a contact into the stage, and `lostOn` triggers the terminal. All-string `stages` keep the legacy money defaults (soldStage = last stage, quotedStage = a stage literally named `"quoted"`); any object entry makes milestones explicit-only. CRM traffic feeds a funnel via composable `bindings` built with `crmPipeline({ provider, pipeline, stages | resolve })` — the declarative `(provider, pipeline)` half is the funnel's traffic claim (exact pipeline beats provider-wide `"*"`; overlapping claims and duplicate ids throw at boot; a funnel's own `"*"` binding doubles as a per-stage fallback for its pipeline-specific maps), and the `resolve` callback form is the escape hatch plugins wrap to own their CRM-speak (outputs runtime-validated; throws are contained). `sources` on the funnel is the ingest-source trust allowlist (mirrors `defineConversion.sources`; browser `inapp` denied by default). Register many via `createHogsendClient({ funnels })`. Deals gain `funnel_id` (nullable, healed on next stage event); `funnel.stage_changed` and the money events carry `funnel_id` as a property so conversion points and journeys scope per funnel via `where` — the money-event NAMES stay stable per funnel-designated milestones. The `crm.{stages,stageMaps}` shorthand is sugar for a single `"default"` funnel (its `stages` accepts the same entries; the standalone `quotedStage`/`soldStage` options are replaced by milestones). Admin deals list/stats/timeseries take a `funnel` param (the default funnel also matches pre-funnel null rows); stats serve the funnel catalog and per-funnel `stageOrder`/`reached`; Studio's revenue dashboard gets a funnel switcher.
