# hogsend

The Hogsend CLI under its bare name — an alias for
[`@hogsend/cli`](https://www.npmjs.com/package/@hogsend/cli) so the obvious
commands just work:

```bash
npx hogsend doctor
pnpm dlx hogsend upgrade
```

Inside a scaffolded app you already have the real thing — `pnpm exec hogsend …`
resolves `@hogsend/cli` from your dependencies.

Docs: [hogsend.com/docs/cli](https://hogsend.com/docs/cli)
